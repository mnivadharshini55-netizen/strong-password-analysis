# Task 6: Password Strength Evaluation

## Objective
To understand how password strength works and evaluate passwords using an online tool.

## Tool Used
- passwordmeter.com

## Passwords Tested
12345 – 10% – Very Weak  
Tamil@123 – 60% – Medium  
T@m1l#2026Secure – 100% – Strong  

## Analysis
Weak passwords like "12345" are easy to crack using brute force attacks.  
Medium passwords like "Tamil@123" are better but still predictable.  
Strong passwords like "T@m1l#2026Secure" are secure because they include uppercase, lowercase, numbers, and symbols.

## Common Attacks
- Brute Force Attack: Tries all possible combinations  
- Dictionary Attack: Uses common password lists  

## Best Practices
- Use 12–16 characters  
- Mix letters, numbers, symbols  
- Avoid common words  
- Use different passwords for each account  
- Enable 2FA

## Conclusion
-Strong passwords improve security and protect user accounts from attacks.
